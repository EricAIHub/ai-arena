## 一、发现了什么？赛道比我们想象的热

调研后发现：**我们不是在做一个冷门产品，而是在一个正在爆发的赛道上。**

### 1.1 直接竞品（已存在）

| 产品 | 定位 | 技术栈 | 亮点 | 缺点 |
|------|------|--------|------|------|
| **Werewolf Arena** (Google Research) | 学术研究，LLM 社交推理评测 | Python + 简单网页 | 首个系统性 LLM 狼人杀评测；Elo 排名 | 纯学术，无 UI，无观赛体验 |
| **Werewolf Bench** (foaster.ai) | LLM 社交智能基准 | Python + 静态页面 | 7 个模型循环赛；分角色 Elo（狼/村民）；市长选举机制 | 无实时观赛，无游戏回放 |
| **MafiaBench** (mafiabench.org) | LLM 社交推理 Benchmark | Python + 简单 UI | 8 人局；2 狼 6 村民；纯文本对战 | 无视觉设计，无互动 |
| **Strategy Bench** (freysa.ai) | 多游戏 AI 排行榜 | Web | 6 种游戏（狼人杀/Secret Hitler/Avalon 等）；欺骗指数+检测指数 | 纯数据展示，无观赛体验 |
| **Wolfcha** (wolf-cha.com) | 用户 vs AI 狼人杀 | Web | 用户当玩家，AI 当对手；有语音 | 不是 AI vs AI 观赛 |
| **AIWolfGame** (GitHub) | 12 人 AI 狼人杀研究 | FastAPI + React | 12 人局（预女猎白）；信息隔离严格；142 个测试 | 研究项目，非产品 |
| **LMSYS Chatbot Arena** | LLM 盲测对比 | Web | 190+ 模型；Elo 排名；100 万+ 投票 | 只有文字对比，无游戏场景 |

### 1.2 间接竞品（AI 竞技/娱乐）

| 产品 | 定位 | 规模 |
|------|------|------|
| **AI Arena (Crypto)** | 区块链 AI 对战，$500K 奖池 | 有博彩，有锦标赛 |

| **MindGames Arena** (NeurIPS) | AI 社交推理竞赛 | 欺骗、谈判、联盟 |
| **Screeps** | AI 编程 MMO | 用户写代码控制角色 |
| **Halite** (Two Sigma) | AI 编程竞赛 | 回合制舰队对战 |
| **SSCAIT** | 星际争霸 AI 锦标赛 | Bot vs Bot，有直播 |

### 1.3 关键发现

**发现 1：AI 辩论在抖音/B站已经火了**
- "AI 撕逼大赛"、"DeepSeek vs Grok 贴吧大战推特" 等视频播放量很高
- 陈铭 vs MiniMax AI 辩论被央视报道
- 用户爱看 AI 吵架，尤其是沙雕话题

**发现 2：学术界在用狼人杀评测 AI 社交智能**
- Google、斯坦福、多个团队都在做
- 核心指标：欺骗指数、检测准确率、投票准确率、存活率
- 这给 AI Arena 提供了"严肃性"背书

**发现 3：Elo 排名是最强的用户粘性引擎**
- LMSYS 靠 Elo 排名吸引了 100 万+ 投票
- 用户每天来看"谁是第一名"，像看体育排名
- 模型排名变化本身就是新闻

**发现 4：观赛体验是最大空白**
- 所有学术竞品都没有好的观赛体验
- 没有弹幕、没有解说、没有回放、没有分享
- **这正是 AI Arena 的机会**

## 三、竞品的优点与我们能学什么

### 3.1 从 LMSYS 学：Elo 排名是王道

**LMSYS 做对了什么：**
- 匿名盲测 → 消除品牌偏见
- Elo 动态排名 → 每天都有新闻（"GPT-5 超过 Claude 了！"）
- 开放数据 → 学术引用 → 权威性

**我们能学：**
- 加入 Elo 排名系统，让每个模型有"段位"
- 盲测模式我们已经有了（✅），但排名还没做
- 排名变化本身就是传播素材

### 3.2 从 Werewolf Arena 学：学术严肃性

**Werewolf Arena 做对了什么：**
- 分角色 Elo（狼人 Elo / 村民 Elo）→ 更精细的模型评估
- 市长选举机制 → 增加博弈深度
- 推理链展示 → 用户可以看到 AI 的思考过程

**我们能学：**
- 分场景 Elo（狼人杀 Elo / 辩论 Elo / 代码对决 Elo）
- 展示 AI 的"内心独白"（推理过程）
- 用学术指标（欺骗指数、检测准确率）增加权威性

### 3.3 从 AIWolfGame 学：工程化

**AIWolfGame 做对了什么：**
- 12 人局，角色完整（预女猎白+警长）
- 信息隔离严格（每个 AI 只看到自己的视角）
- 142 个测试用例
- JSONL 日志 + 人类可读的编年史

**我们能学：**
- 增加更多角色（猎人/女巫/守卫/警长）
- 严格信息隔离（当前我们的 AI 可能看到过多信息）
- 生成人类可读的"游戏编年史"（回放文本）

### 3.4 从抖音/B站学：内容传播性

**AI 辩论视频为什么火：**
- 沙雕话题（"猫和路由器哪个更重要"）
- AI 的"人格化"（DeepSeek 的毒舌、GPT 的圆滑）
- 对抗性（两个 AI 互怼，观众站队）
- 短（1-3 分钟精华）

**我们能学：**
- 沙雕辩论场景是最高优先级
- 每局自动生成"精彩语录"卡片
- 支持 1-3 分钟的"精华回放"导出

### 3.5 从 Strategy Bench 学：多维度评估

**Strategy Bench 做对了什么：**
- 6 种游戏综合排名
- 欺骗指数（当狼人时的伪装能力）
- 检测指数（当村民时的推理能力）
- 按游戏类型分维度评估

**我们能学：**
- 不只看"赢了没有"，还要看"怎么赢的"
- 评估维度：推理能力、欺骗能力、说服力、存活能力
- 生成"AI 能力雷达图"

## 四、我们的差异化优势

竞品调研后，AI Arena 的独特定位更清晰了：

| 维度 | 学术竞品 | AI Arena |
|------|----------|----------|
| **定位** | Benchmark / 研究工具 | **观赛平台 / 娱乐产品** |
| **用户** | 研究者、开发者 | **普通用户、内容创作者** |
| **体验** | 纯文本、无设计 | **苹果级 UI、沉浸式观赛** |
| **场景** | 只有狼人杀 | **5 个场景，可扩展** |
| **模型** | 固定几个模型 | **用户自选任意模型** |
| **互动** | 无 | **盲测、竞猜、弹幕（规划中）** |
| **传播** | 无 | **分享卡、短视频（规划中）** |

**一句话：学术界在做"AI 能不能玩狼人杀"，我们在做"AI 玩狼人杀给你看"。**

## 五、战略调整建议

基于调研，原路线图需要调整优先级：

### 调整 1：Elo 排名系统提到最高优先级

**原计划**：P2（中期）  
**新计划**：P0（立即）

**理由**：
- LMSYS 靠 Elo 排名吸引了 100 万+ 投票
- 排名变化本身就是新闻（"DeepSeek 狼人杀 Elo 超过 GPT 了！"）
- 没有排名，用户看完一局就走了；有排名，用户每天来看

### 调整 2：沙雕辩论场景提到最高优先级

**原计划**：P1（短期）  
**新计划**：P0（立即）

**理由**：
- 抖音/B站已经证明 AI 辩论有传播性
- 沙雕话题（"WiFi 和空调只能留一个"）天然适合短视频
- 比狼人杀更容易理解，门槛更低

### 调整 3：增加"AI 能力评估"维度

**原计划**：无  
**新计划**：P1（短期）

**理由**：
- 学术界用欺骗指数、检测准确率等维度评估 AI
- 这些指标给产品增加"严肃性"和"权威性"
- 用户不只是看热闹，还能看门道

### 调整 4：增加"游戏编年史"功能

**原计划**：无  
**新计划**：P1（短期）

**理由**：
- AIWolfGame 的"人类可读编年史"是很好的设计
- 每局结束后生成一段文字叙述，像体育解说回放
- 这是传播素材，用户可以直接复制发朋友圈

### 调整 5：短期不做房间系统

**原计划**：P3（长期）  
**新计划**：暂不规划

**理由**：
- 竞品调研显示，用户更在意"内容"而非"社交"
- 先把单人观赛体验做到极致，再考虑多人
- 房间系统工程量大，收益不确定

## 七、关键结论

1. **赛道确认**：AI 对战/社交推理评测是 2025-2026 年的热点赛道，学术界和产业界都在做
2. **空白确认**：所有竞品都缺"观赛体验"，这是 AI Arena 的核心差异化
3. **传播性确认**：AI 辩论视频在抖音/B站已经火了，沙雕场景是流量密码
4. **Elo 排名是王道**：LMSYS 证明了排名系统是最强的用户粘性引擎
5. **短期目标调整**：Elo 排名 + 沙雕辩论 + 游戏编年史 → 三板斧打出去

**一句话：不要做第 N 个 AI Benchmark，要做第一个 AI 观赛娱乐平台。**
