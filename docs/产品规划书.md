## 🎯 目标用户画像

**核心用户**: 18-30 岁，对 AI 感兴趣的技术/泛技术人群

| 用户类型 | 特征 | 需求 |
|----------|------|------|
| **AI 爱好者** | 用过 ChatGPT/DeepSeek，好奇不同模型差异 | 想直观对比哪个 AI 更强 |
| **内容创作者** | B站/抖音/小红书 UP 主 | 想做"AI 对战"类视频内容 |
| **开发者** | 程序员，关心技术实现 | 想测试自己的模型/API |
| **学生** | 大学生，对 AI 好奇 | 课余娱乐、社交分享 |
| **科技博主** | 写评测、做分析 | 需要素材和数据 |

## 🎨 用户喜欢什么样的产品

### 从爆款项目学到的

| 项目 | 为什么火 | 我们学什么 |
|------|---------|-----------|
| **Mafia Boardgame** (6⭐) | AI 狼人杀，有性格，有实时对话 | 场景要有趣、AI 要有"人设" |
| **Quorum CLI** (107⭐) | 多模型辩论，CLI 简洁 | 功能要聚焦、不要过度设计 |
| **MoneyPrinterTurbo** (92K⭐) | 一键生成视频，开箱即用 | 用户体验要极致简单 |
| **HivisionIDPhotos** (17K⭐) | 证件照，免费，有 Docker | 解决真实痛点 + 免费 |

### 用户心理

1. **"我要一键就能用"** — 不要让我配置半天
2. **"我要好看"** — 界面丑的工具我不用
3. **"我要分享"** — 能截图/录屏发朋友圈/B站
4. **"我要免费"** — 收费的我不考虑
5. **"我要有趣"** — 工具可以，但最好有娱乐性

## 🏗️ 产品架构决策

### ✅ 决定：一个安装包 + 内置多主题

**理由**:
1. 用户不想下载多个版本
2. 主题切换是"高级感"的体现
3. 维护成本低（一套代码）
4. 安装包大小影响不大（CSS 文件很小）

### 主题设计

| 主题 | 风格 | 目标用户 |
|------|------|---------|
| **纯净** (默认) | 极简深色，Linear 风格 | 开发者、喜欢简洁的人 |
| **豪华** | 渐变光效，赛博朋克 | 年轻用户、想要酷炫感 |
| **柔和** | 暗灰+暖色点缀 | 长时间使用、护眼 |

### 安装包形态

```
AI-Arena-Setup.exe (~50MB)
├── Electron 桌面壳
├── Python 后端
├── 前端界面（含 3 个主题）
├── 5 个预设场景
└── 一键启动
```
